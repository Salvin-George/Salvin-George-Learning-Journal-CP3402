## Content Insights

- This week helped me relearn the difference between HTML, CSS, and JavaScript. I re-learned that HTML should focus on structure and meaning, while CSS controls presentation and layout, and JavaScript is responsible for behaviour and interactivity. Using Bootstrap made this separation clearer, as many layout and styling decisions are handled through predefined classes rather than custom CSS.
